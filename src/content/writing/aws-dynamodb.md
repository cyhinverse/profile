# Amazon DynamoDB: Serverless NoSQL for Speed

**Amazon DynamoDB** is a [fully managed](#green), [serverless](#green), [key-value NoSQL database](#purple) designed to run [high-performance applications](#yellow) at [any scale](#yellow).

### What is it?

Unlike relational databases ([SQL](#purple)), DynamoDB is designed for [non-relational data](#green) where [speed](#yellow) and [scalability](#yellow) are paramount. It [automatically manages](#underline) data traffic and storage capacity.

### Key Features

- [Performance](#green): Consistently delivers [single-digit millisecond response times](#yellow).
- [Serverless](#green): [No servers to provision or manage](#underline).
- [Scalability](#green): [Virtually unlimited throughput and storage](#yellow). It scales [horizontally](#purple).
- [Integrated](#green): Built-in [security](#purple), [backup and restore](#purple), and [in-memory caching](#purple) ([DAX](#purple)).

### Use Cases

- Mobile backends.
- Gaming (player profiles, leaderboards).
- Ad tech (storing user impressions).
- IoT (Internet of Things) functionality.
- Shopping carts (high-volume e-commerce).

### Why use it?

When your application needs to handle [millions of requests per second](#yellow) with [ultra-low latency](#yellow), [DynamoDB](#purple) is the [go-to solution](#underline).
