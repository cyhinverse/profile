# Amazon S3: The Storage Backbone of the Internet

**Amazon Simple Storage Service (Amazon S3)** is an [object storage service](#green) offering industry-leading [scalability](#yellow), [data availability](#yellow), [security](#yellow), and [performance](#yellow).

### What is it?

S3 stores data as "[objects](#green)" within resources called "[buckets](#green)". Itâ€™s like a [massive, infinite hard drive](#underline) in the cloud where you can store any file type: [images](#purple), [videos](#purple), [logs](#purple), [backups](#purple), etc.

### Key Features

- [Durability](#green): Designed for [99.999999999% (11 9's)](#purple) of [data durability](#yellow). Your data is [redundant](#yellow) across [multiple facilities](#underline).
- [Scalability](#green): Store and access [any amount of data](#underline).
- [Security](#green): Supports [encryption](#purple) (at rest and in transit) and [fine-grained access controls](#purple).
- [Cost-Effective](#green): [Tiered storage options](#yellow) ([Standard](#purple), [Intelligent-Tiering](#purple), [Glacier](#purple)) help you [optimize costs](#yellow) based on access frequency.

### Use Cases

- Backup and restore.
- Disaster recovery.
- Cloud-native application data (hosting static websites).
- Data lakes for big data analytics.

### Why use it?

It is the [most popular choice](#yellow) for [cloud storage](#green) due to its [extreme durability](#yellow) and [integration](#yellow) with almost every other [AWS service](#purple).
