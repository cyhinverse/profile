# Amazon EC2: Your Virtual Server in the Cloud

**Amazon Elastic Compute Cloud (Amazon EC2)** is a [web service](#green) that provides different types of [resizable computing capacity](#yellow) in the cloud. It is designed to make [web-scale cloud computing](#purple) easier for developers.

### What is it?

Think of EC2 as a [remote computer](#green) that you can rent. You can choose the [CPU](#purple), [RAM](#purple), and [Storage](#purple) capacity you need, and you [only pay for what you use](#underline).

### Key Features

- [Scalability](#green): Launch as many or as few [virtual servers](#purple) as you need. Scale [up or down](#yellow) within minutes.
- [Control](#green): You have [complete control](#yellow) over your instances, including [root access](#underline) and the ability to interact with them as you would any machine.
- [Flexibility](#green): Choose from multiple operating systems ([Linux](#purple), [Windows](#purple)) and [instance types](#purple) tailored for different use cases ([Compute Optimized](#purple), [Memory Optimized](#purple), [GPU](#purple), etc.).
- [Reliability](#green): Amazon offers a [99.99% availability SLA](#yellow) for each EC2 region.

### Use Cases

- Hosting websites and web applications.
- Running enterprise applications (SAP, SharePoint).
- High-performance computing (HPC).
- Deep learning and machine learning training.

### Why use it?

It [eliminates the need to invest in hardware](#underline) specifically, so you can develop and deploy applications [faster](#yellow).
